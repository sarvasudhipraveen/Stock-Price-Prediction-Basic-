# Stock-Price-Prediction-Basic-
Stock Price Prediction Using Linear Regression
Project Overview

This project demonstrates the basics of Time Series Forecasting using historical stock market data. The objective is to predict future stock prices using the Linear Regression algorithm and understand fundamental concepts of stock price prediction.

Objective
Load and visualize historical stock price data.
Perform basic data preprocessing.
Split data chronologically into training and testing sets.
Train a Linear Regression model.
Predict future stock prices.
Evaluate model performance.
